### 配置格式化

配置 Prettier 为默认代码格式化工具

- 同时按下 command/ctrl + shift + p 键盘
- 输入 format document with
- 选择 Configure Default Formatter…
- 选择 Prettier 为默认的工具

配置文件保存时自动格式化 setting

![image-20230716235928167](img/快捷键及操作.assets/image-20230716235928167.png)
