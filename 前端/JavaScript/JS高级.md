**垃圾回收机制**

内存的分配和回收

全局变量一般不回收（关闭页面回收）；局部变量不用了就会被回收；

内存泄漏：程序中分配的内存由于某种原因程序未释放或无法释放叫做内存泄漏

闭包 = 内层函数 + 外层函数的变量

闭包存在的问题：内存泄漏

```js
  // 闭包实现数据的私有
  function count() {
    let i = 0;
    function fn() {
      i++;
      console.log(i);
    }
    return fn;
  }
  const fun = count();
```
