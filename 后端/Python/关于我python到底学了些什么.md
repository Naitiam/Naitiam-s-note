## 运算符


| 运算符 | 说明 | 实例 | 结果 |
| ----- | ---- | ---- | ---- |
| `+`    | 加 | 12.45*15 | =27.45 |
| `-`    | 减 | 4.56-0.26 | =4.3 |
| `*`    | 乘 | 5*3.6 | =18.0 |
| `/`    | 除 | 7/2 | =3.5 |
| `//`   | 整除 | 7/2 | =3 |
| `%`    | 取余 | 7%2 | =1 |
| `**`  | 幂运算 | 2**3 | =8 |
